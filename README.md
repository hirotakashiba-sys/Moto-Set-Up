# 🏍 Moto Set Up (モト・セットアップ)

Moto Set Upは、モトジムカーナ競技者向けの高度なモーターサイクル・セッティング＆シミュレーションツールです。
複雑な車体ディメンション（ジオメトリ）の動的変化を手のひらで可視化し、日々の練習記録とタイム比を管理することで、勘に頼らないロジカルなマシンセットアップをサポートします。

## ✨ 主な機能 (Features)

### 1. 📐 動的ジオメトリ・シミュレーション
突き出し、車高調整、プリロードなどの変更から、走行時の車体姿勢をリアルタイムに計算します。
* **動的トレール解析:** 減速Gに応じたフロントフォークの沈み込み量を計算し、ブレーキング時の実効キャスター角とトレール量を算出・グラフ化（`fl_chart`）します。
* **アンチスクワット (AS) 解析:** スプロケット丁数、スイングアーム長、ピボット高、重心位置から、加速時のAS率、回生時のEB率、制動時のAL率をストローク量ごとに計算して可視化します。
* **サグ自動計算:** 0G, 1G, 1G'の数値を入力するだけで乗車Sagを自動算出します。

### 2. ⏱️ タイム＆履歴管理
* **TOP比の自動計算:** 練習会のトップタイムと自分のタイムを入力し、タイム比（%）を自動計算します。目標タイム（105%など）の逆算機能も搭載。
* **場所別セッティング履歴:** サーキットや練習会場ごとのセッティングデータを保存。場所ごとのフィルタリング表示が可能です。
* **相関グラフ機能:** 過去の履歴データから、「トレール量 vs TOP比」「AS率 vs TOP比」などの相関関係を散布図としてプロットし、自分にとって最適な数値を導き出します。

### 3. 🤖 AI連携 & トラブルシューティング
* **AIセッティング・アドバイザー:** 「アンダーステア」「切り返しが重い」などの症状を選択すると、現在のジオメトリ計算結果に基づいて具体的な調整案を提示します。
* **AIエンジニアへの依頼書生成:** 現在のセッティングデータ、コンディション、インプレッションをまとめたLLM（ChatGPTやClaude等）向けのプロンプトをワンタップでクリップボードに生成します。

### 4. ☁️ 天気・コンディション自動取得
* **Open-Meteo API連携:** 登録した走行場所の住所（緯度・経度）から、当日の「気温」「湿度」を自動取得。さらに降水量から「ドライ/ウェット」を、気温から「路面温度」を推計して自動入力します。

### 5. 💾 ローカルデータ管理
* オフライン環境（山奥の練習会場など）でも動作するよう、データはすべてデバイス内（`SharedPreferences`）に保存されます。
* JSONテキスト形式でクリップボードへExport/Importでき、機種変更時の移行やデータのバックアップが容易です。
